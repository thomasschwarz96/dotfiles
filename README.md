# Dotfiles
My personal environment

## Installation
If you want to install my `.dotfiles` just clone it and run the `setup.sh` file.

## Required
- `i3`
- `i3status`
- `dmenu`
- `kitty`
- `tmux`
- `vim`
- `git`
- `ranger`
- `compton`
- `feh`

## Optional, not required
- `Enpass`
- `neix`
- `xfce4`
    - Used for some programms
